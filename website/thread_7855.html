【短篇】音之風 第七章 風雲一賦始天涯

一處殿堂，一處深殿，一襲紫杉，一個冷笑，那人，拿起龍頭魔杖，放聲一笑：「音天城一旦破了，三界就歸我統屬，sword，我到要看是誰先笑了出來。」語畢，那人伴隨著腳步聲，一步一步地消失了。

「呼，這什麼惡夢。」sword反身起床，如今他修過流冰玄門正宗的內功，睡覺也是打坐著睡，不過他醒來第一件事，就是走去殿堂，問正在吃早餐的tails說：「昨天道長劈開sonichub的通道，成功了嗎？」tails擺擺手：「嘆，昨日道長行十劍破空陣，說是劈開了一點點，內息不順，這才罷手。」「早跟他說不用這樣了，那後來呢？」tails接著說：「現下道長已和眾人討論過，說是將七劍置於城門外，置一個『天罡北斗陣』，以七劍之力護住城門，如果那禍要來……。」這時，流冰道長走了上來，拱起手來向sword和tails說：「劍兄早，那……不知狐狸兄如何稱呼？」sword哈哈一笑：「算了，沒關係，道長養傷去吧，順便去和景兄下下棋吧。」流冰拱了拱手，走進房間。

Tails悄悄的對sword說了幾句：「真怪，道長一點傷也沒有。」sword說：「不對，劈空之傷乃是內傷，但道長不可能走路還如此端正呀？」

( 在音天城的書房 )

Sonic四處翻了翻書櫃，knuckles問道：「你這樣翻箱倒櫃，就不怕sword罵人嗎？」sonic回嘴說：「算了吧，sword脾氣很好，就算被他發現，就說我在學中國字不成？」knuckles想了想，也不無道理。

這時，一處縫隙中掉出一本書，sonic拿起來，拍了拍上面的灰塵，只見上面寫著「風雲訣」，sonic好奇，才剛翻了翻，忽然一道黑影飛過，劍光一閃，sonic急忙閃過，反手一拔，使出「音速劍法」，但那人只拆了幾招，反手一抓，抓住sonic的劍柄，喝了一聲：「為甚麼看我的書？」sonic定睛一看，是sword！sword怒氣沖沖的質問：「為什麼看這本書？」sonic也只是因為一時好奇才看的，但仍然不敢辯解，sword嘆了口氣，收回劍：「方才沒有受傷吧，小弟脾氣躁些，不喜歡他人翻我此書，還望各位見諒。」sonic看著sword走了出去，覺得sword絕對跟此書有甚麼心結。

恰巧amy走進來，一直找不到sonic，正巧看見sword走出來，還道他是sonic，雙手緊緊的抱住他，sword一驚，喝道：「不要這樣！」，反手拔劍，對著amy狠狠地說道：「想做甚麼，害死我嗎！」amy本就被sword冷冷的聲音嚇到，加上sword的斥責，忍不住掩著面嗚嗚嗚的就哭了起來。Sonic聽到哭聲，三步併作兩步的跑了進來，扶著amy，但才轉頭，人就不見了。


「說起來，也是貧道的錯。」一個聲音從走廊上傳來。

Sonic問道：「道長，sword怎麼會突然抓狂呢？」流冰嘆道：「當初傳給他的那股內力，乃是以『魔訣寶法』煉製而成，持此內力者不可被情感所動，否則內功逆行，輕者吐血，重者殘疾！」過了不久，果然聽到sword傳來的叫聲：「我的眼睛，我的眼睛！」眾人不禁駭然，amy想去道歉，但想起道長說的「不可被情感所動」，還是算了。

此時，一位衛兵衝了進來：「報，魚腸劍陣被毀了，七大神劍陣只剩六個呀！」「甚麼？」

( 夜深人靜之時…… )

夜黑風高，燭火搖曳，流冰坐在桌子旁邊，拿出了七塊板子，伴隨著燭火的燈光，拿起一塊寫著「魚腸」的牌子，手指輕輕一碰，那板子霎時就灰飛煙滅，流冰自言自語的說道：「我下一步該怎麼做呢？」旋即放聲大笑，不過，除了他以外，沒有任何人聽見這陰沉詭譎的笑聲了。
