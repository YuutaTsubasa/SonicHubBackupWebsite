👤 aoifh 🕐 2009-05-20 04:28:17
可以不用HBC

1. 先下載相關的檔案，並將它解壓縮。

2. 在你的主機上，把你的遊戲 Save 檔 copy 一份到 SD Card 上。
(每個遊戲有其特定的目錄，所以，要跟著其目錄結構放 Save 檔)

3. 用讀卡機將 SD Card 連上 PC，將下載來的 Save 檔 (data.bin) 蓋掉SD card特定的子目錄下的data，如SBK的目錄就是 RENE。

4. 複製好後，將 SD Card 插回主機去。

5. 先把主機上該遊戲的 Save 紀錄刪除 (一定要做)

6. 然後再把 SD Card 上的遊戲 Save 檔複製一份到主機上。
