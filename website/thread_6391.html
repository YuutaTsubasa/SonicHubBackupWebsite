Sega遭駭客入侵 盜走130萬客戶資料

繼索尼及任天堂後，日本電腦遊戲軟件開發商世嘉 (Sega) 亦受到駭客入侵， 130 萬客戶的資料被盜取。用戶的姓名、電郵地址、出生日期及加密密碼等個人資料外洩，所幸處理信用卡號碼的外部支付系統安全，用戶的財務資料沒有外洩。

《星島日報》報導，世嘉周日 (19日) 發表聲明指出，網上資料庫 Sega Pass System 中有 130 萬英語用戶的資料，經世嘉歐洲分部 Sega Europe 的網站被盜資料。世嘉在發現問題後，已於本月 16 日即時關閉有關資料庫，並發出電郵通知用戶。

世嘉指出，用戶姓名、電郵地址、出生日期及加密密碼遭黑客竊取，但強調沒有對用戶帶來任何財務風險。 Sega Europe 的線上網絡主要為登記客戶提供新產品資訊。

世嘉表示，公司就事件對顧客帶來麻煩深感抱歉，並已經交由專家調查今次黑客入侵事件。而 Sega Pass 目前已經離線，未悉何時可以恢復，而用戶將要重新設定密碼。

該公司同時強調，帳戶密碼均以加密形式儲存，而且信用卡資料並沒有儲存在世嘉的伺服器上，因此仍然處於安全狀態。

不過，該公司建議用戶盡快修改登入的個人資料，同時應小心以 Sega Pass 名義所發出電郵的真確性，以防駭客採用「網絡釣魚」的欺詐手法。

此外，開發「超音鼠」等著名遊戲軟件的世嘉表明，由於沒有個人付款的資料儲存於公司伺服器內，而是採用第三方的付款方式，因此駭客入侵事件並不會對用戶造成金錢上的直接損失，因此用戶毋須就今次事件停用信用卡。

外界就世嘉在事發後 24 小時內即公開被駭客入侵，作出高度評價，指相較索尼 (Sony) 事發超過一周才公開事件，世嘉的做法更符合企業責任，而且公開的事發內容詳細而透明。

最近一個月， Bethesda、Codemasters、Epic Games、任天堂、 Sony 等遊戲廠商均受到駭客攻擊，矛頭直指名為 Lulz Security 的駭客組織，但該組織聲稱這次事件與之無關。
