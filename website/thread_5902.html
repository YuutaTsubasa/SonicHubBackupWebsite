Sonic Colors Wii GameLand的小秘密

👤 RedFox 🕐 2010-12-06 10:49:35
Game Land，據某蛋的說法：一個和你那些可悲朋友浪費寶貴時間的好地方
不管那傢伙怎麼說，在Game Land除了拿取Chaos Emerald以外，還有一些樂趣可以發掘喔
其實以下的內容說明書上幾乎都有寫，不過我知道很多人手邊是沒有那玩意的，所以還是貼上來分享
以下操作按鍵的排序為（WiiRemote + Nunchuk / WiiRemote橫拿 / Wii Classic Controller / Game Cube Controller）

選擇人物畫面
如果你已經有30個Special Ring，可以開啟Mii（這大家應該都知道...），某狐認為這實在是電玩界裡非常詭異的畫面之一......
試著按下其他選擇 / 退回以外沒有實際功能的按鍵，Sonic的顏色會有變化喔！
SPEED有1顆星到5顆星，代表遊玩速度，可以根據玩家的技術和習慣調整
LIMITER是限速器，處於ON狀態的話，系統會壓抑跑在前面的人的速度

1 Player
如果你真的找不到其他的伴陪你浪費時間一起遊玩，那就只好一個人進行啦
不過一個人進行還是有個小技巧可以使用，按下你的2 / A / y / Y，可以讓你跟你背後那顆「球」切換過來
有什麼用呢？兩邊的Boost能量和Color Power是分開存放的，這表示你有兩個Color Power可以運用囉

1.5 Players
一個半的玩家聽起來還蠻恐怖的，實際的意思是像Sonic 3一樣
一個人為主一個人為輔的玩法，輔助的人不會掛掉，死了會回歸P1的身邊，按一下就可以復活
Boost能量和Color Power是共用的，適合兩個實力相差太大的人一起玩

2 Players
重點來了，本模式本來就是設計給兩個人遊玩的
所以選擇這個模式的話，會比前兩個模式有更多好康可以用

Color Power感染
如果玩家其中一方發動Color Power，再去接觸了另一玩家即可發動Color Power感染
被接觸的人也會立刻得到該力量，並發動使用

協助追尾模式
按下2 / A / y / Y，你控制的角色便會進入Spin狀態，跟著另一人跑，就跟死掉重生時的狀態一樣
再按一下2 / A / y / Y即可恢復，不過要是這時另一玩家掛了，那就得重來了...

拉畫面
兩人進行時，落後的一方會被前面的人的畫面推著走
這時如果碰到牆壁無法前進而被拉出畫面，那就會死掉
這種情形發生時，畫面上會出現一個箭頭圖示，落後的人只要這一瞬間按下箭頭的方向
就可以把畫面和另一玩家拉回來，躲過一死

Color Power合體
不用多說，兩個人同時發動Color Power即可
這時兩股力量會結合，形成更強的威力！

撞擊球
就是那些在關卡中用一條細線吊著的星星球，撞了會噴出少得可憐的1 Ring
其實，如果有兩個玩家分別從兩側同時撞擊，那它給出的Ring就不是少少的一兩個囉......
