NVIDIA 3 卡 SLI 正式上路

👤 opus 🕐 2007-12-14 12:57:21

有三條 PCIe x16 插槽的主機版我們已經看得久了（好吧，也沒那麼久啦...），而現
在能全面利用這多出來的 PCIe 的技術終於現身了：NVIDIA 的 3 卡 SLI。目前支援 3 卡 SLI 的 GPU 只有 GeForce 8800GTX 和 GeForce 8800 Ultra 兩個，所以組一台 3 卡 SLI 的系統不論在 $$ 還是電力上的花費都會非常驚人。Hothardware.com 入手了一組 3 SLI 系統，而毫不誇張的說，電腦電源打開的那一瞬間整個房間的燈都暗了下去。


有趣的是，在 1920x1200，不開 FSAA 的這種「普通」解析度下，3 卡 SLI 的表現比普通的雙卡 SLI 還要差 -- 主要是因為多的運算力無處可使，還要多花資源做負載平衡。一直到 2560x1600 / 8xAA / 16x Aniso 這種瘋狂的等級，才能看出 3 卡 SLI 大約有單卡 2.8x 左右的效能。
