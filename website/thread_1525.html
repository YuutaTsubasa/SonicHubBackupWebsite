Windows XP SP3已排入Windows Update發布

👤 opus 🕐 2008-05-07 02:47:44
Windows XP SP3已排入Windows Update發布，Vista SP1則要再等等喔！


等很久了吧，微軟Windows的奴隸與粉絲們，國外現在已可從Windows Update找到Windows XP SP3的選項，終於不用再去網路上「低調」這些更新，還要擔心會不會被人偷加「加料」。

在經過國外志願試用者測試過後，Windows XP SP3（感謝
windoze提醒）終於決定正式發布更新，已經可以透過Windows Update來進行，更新後號稱可以提昇10％的效能，但也帶來uxtheme.dll與TCP/IP連線數的限制，不過繁體中文版到完稿時仍看不到Windows Update出現SP3的選項。

至於Windows Vista SP 1，由於國外試用者發現會與自家的Dynamics RMS（Retail Management System）有著相容性上的問題，因此暫緩正式發布的動作。
