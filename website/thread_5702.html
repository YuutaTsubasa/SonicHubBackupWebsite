SEGA經典攜帶型遊樂器《GAME GEAR》二十周年紀念專題！

👤 aoifh 🕐 2010-10-09 01:21:58
家用遊樂器市場曾經有過幾次的王座交替，不過在攜帶型遊樂器市場，市場龍頭的唯一獨佔者一直都是任天堂，從初代GAMEBOY面市到現在這二十多年間，在攜帶型遊樂器市場向任天堂發出挑戰的公司並不在少數，但大多數的挑戰者都只能夠撐個短短一兩年就鎩羽而歸，能夠一直和任天堂周旋著，維持著多年軟硬體生命週期的競爭者，除了近年SONY的PSP以外，另一個能夠力戰到底的，就是SEGA的GAME GEAR，2010年10月6日是GAME GEAR上市二十周年的日子，就讓我們來緬懷這台陪伴很多玩家度過歡樂時光的主機吧！

任天堂在1989年4月發售的GAMEBOY以可替換卡匣遊戲的方式，將家用遊樂器的運作模式帶到攜帶型遊樂器上，為電玩產業創造了一個全新的市場，在當時，做為任天堂在家用遊樂器市場主要競爭者的SEGA，也決定要投入攜帶型遊樂器市場，推出名為GAME GEAR的攜帶型遊樂器，全方面的對任天堂宣戰。

在家用主機市場與任天堂已交手數次的SEGA了解，如果是單純要比遊戲數量的話，SEGA方面是處於弱勢的，想要勝過任天堂，則必定要有一些決定性的亮眼之處，於是，針對GAMEBOY的四段灰階反射式螢幕，SEGA讓GAME GEAR搭載了彩色背光螢幕，並針對這點差異進行強力的比較性宣傳廣告，企圖以強烈的畫面表現差異來挑戰任天堂。

GAME GEAR的硬體架構是直接承襲自SEGA當時1985年推出的家用主機SEGA MARK-III，在CPU、記憶體、音效能力等各方面是完全相同，僅僅降低了畫面解析度，不過卻有著更高的調色盤總數與同時發色能力，雖然當時家用遊樂器市場的主流機種已經步入16位元時代，SEGA MARK-III已經是上個市代的產品，但對於剛起步的攜帶型遊戲器主機來說，GAME GEAR在硬體效能上真的是非常強悍。

為了進一步凸顯出彩色螢幕的優勢，SEGA更為GAME GEAR設計了專屬的電視訊號接收器(TV Tuner)，讓GAME GEAR不但可以玩遊戲，也可以化身為隨身電視隨時隨地觀賞，甚至TV Tuner上面還具備了AV端子輸入孔，可以和其他影音設備相互搭配使用，大幅提升泛用性。
[附件 4365 未找到]

1990年10月6日，GAME GEAR正式在日本上市，打著日本首款使用彩色螢幕的攜帶型遊樂器名號強力挑戰任天堂GAMEBOY，還刻意使用針對性的廣告，強調彩色與黑白的不同。


不過彩色螢幕並非SEGA的專利，在GAME GEAR之前，ATARI就已經在美國推出彩色螢幕的攜帶型遊樂器LYNX了，GAME GEAR上市兩個月後，NEC也把家用遊樂器PC-Engine直接縮小化，推出軟硬體與家用機完全相容的攜帶型遊樂器PC-Engine GT，面對這些同為彩色螢幕、性能也不遜色的同質競爭者，GAME GEAR倒是成功的以較低的價格與較豐富的軟體陣容從其中脫穎而出，成為當時代表性的彩色螢幕攜帶型遊樂器。

但就算GAME GEAR能夠勝過其他幾台彩色螢幕攜帶型遊樂器，真正需要面對的對手還是任天堂的GAMEBOY，而這些使用彩色螢幕的攜帶型遊樂器，對上了GAMEBOY即使能夠在畫面表現上佔上風，但以「攜帶型遊樂器」這樣的產品定位來說，卻有著極端不利的普遍狀況。首先是價格的高昂，GAMEBOY的售價僅12800日圓，對比之下LYNX要29800日圓，PC-Engine GT更是44800日圓的天價，光是價格上的劣勢後兩者就差不多可以提前出局了，只有GAME GEAR硬是將價格壓低到兩萬元以下的19800日圓，才保有一點價格競爭力。

受限於當時的技術，彩色液晶螢幕沒辦法像灰階液晶螢幕那樣使用反射光源的方式成像，而必須依靠側光或背光等主動式光源，這不但導致主機的體積變得厚重，更有著非常誇張的耗電量，不論是哪台彩色螢幕主機，一次都要用上六顆AA電池才能夠驅動，而且電池的續航力頂多只能撐個兩三小時就消耗殆盡，這對於強調攜帶性、適合外出遊玩的攜帶型遊樂器來說絕對是非常嚴重的致命傷，尤其對比上GAMEBOY僅需四顆AA電池就可以運作三十多小時的省電性，硬體層面的優勢反倒諷刺的成了最大的痛處。

嚴格來說，在日本地區GAME GEAR的銷售表現不能稱得上是亮眼，從1990年10月上市後，一直到1993年7月才終於達成一百萬台的銷售量，不過跟其他同期彩色螢幕主機墳前早就長滿草的慘況比起來，GAME GEAR也算是很努力的支撐下來了。

GAME GEAR還有一個特色是SEGA在中期開始採用較為明亮、年輕化的形象來行銷，這跟SEGA系主機過去總是灰黑色、高年齡玩家取向的既有印象不同，GAME GEAR發行了在當時相當少見的多種彩色機殼主機、並請來了當紅偶像明星高橋由美子代言，獲得了不錯的迴響。


即使在日本GAME GEAR發展得不算特別成功，但是在歐美市場，嘿，這可剛好是SEGA最活躍的一段時代，靠著家用主機GENESIS(海外版MEGA DRIVE)的熱賣與任天堂分庭抗禮，SEGA藉由將GENESIS招牌遊戲移植或製作衍伸作品在GAME GEAR上，成功的為GAME GEAR凝聚不少買氣。

北美SEGA最擅長的諷刺廣告也用到了GAME GEAR上面，將黑白的GAMEBOY與GAME GEAR直接做了跟尖銳的對比：




GAME GEAR在歐美的銷售情況相當不錯，此情況還跟著提攜了SEGA MASTER SYSTEM(海外版SEGA MARK-III)這台主機。由於硬體規格幾乎相同，兩者間的遊戲可以很輕鬆的相互移植，雖然MASTER SYSTEM在當時已經是退役機種，不過在中南美、東歐等開發中國家，MASTER SYSTEM還屬於主流遊戲機，對新軟體的有一定的需求量，這時與GAME GEAR遊戲的移植就成了最方便的遊戲提供方式，這種在不同等級市場間相互共享資源的例子並不能說多見，卻有效的相互延長了彼此的軟體生命週期。

GAME GEAR一路與GAMEBOY鏖戰了超過五年，到了1996年時，SEGA為了將資源專注投注在家用主機SEGA SATURN上，決定中止MEGA DRIVE遊戲的開發，GAME GEAR方面，則將其移交給玩具事業部(現在的SEGA TOYS)。玩具事業部將GAME GEAR轉型成低年齡取向，以「KID\S GEAR」為名發售主機與遊戲，「KID\S GEAR」主機只是原有的GAME GEAR小幅改款，按鈕造型修改並在機殼印上了動畫版《VR快打》的結城晶圖像。
[附件 4367 未找到]

將GAME GEAR的經營移交給非主力的玩具部門，其實也就代表了SEGA已經無心無力再繼續發展GAME GEAR了，1996年當時，由於32位元次世代主機為遊戲的聲光效果帶來極強烈的刺激和改變，既有的攜帶型遊樂器已然失去吸引力，市場衰退的情況就連任天堂也有些無能為力，1995年嘗試推出的VIRTUAL BOY不但沒辦法承接GAMEBOY的地位，市場反應更是悽慘的完全失敗，對任天堂和SEGA來說，攜帶型遊樂器市場都是差不多該收手了。

於是，轉型成KID\S GEAR的GAME GEAR已經不再是公司的主力商品，軟體的供應在1996年末正式畫下句點，並且沒有推出後繼機種的計畫，讓GAME GEAR成為SEGA唯一一台攜帶型遊樂器機種。不過GAME GEAR倒也難能可貴的成為SEGA歷代眾主機中，生命週期較長的一個，其全球累計一千四百萬台的銷售量，也僅次於MEGA DRIVE(GENESIS)，寫在SEGA的歷史中依然可算是一頁輝煌。

至於任天堂方面呢？GAMEBOY的軟體供應原先也差不多就要在1996年中終止，只是沒有人會預測到，當年排在GAMEBOY發行表上倒數幾款，彷彿關門人一樣的最末期軟體，化身成了甦生術，救活了GAMEBOY，讓GAMEBOY以空前的奇蹟般姿態在市場上復活...那款遊戲，名為《ポケットモンスター》，《神奇寶貝》...。

GAME GEAR的遊戲主要有三種開發方向，一種是移植作品，移植自MARK-III或者MEGA DRIVE，一種則是以既有的名作標題，在GAME GEAR上開發衍伸的外傳作品，最後則是完全針對GAME GEAR開發的原創作品，雖然遊戲發片的速度並不快，主要的軟體量也是由SEGA一家在支撐，不過其中的高品質遊戲並不算少數，有著不少表現優異的作品，以下簡單列舉一些代表性作品：

曾任SEGA社長的佐藤秀樹曾經在多年後說明GAME GEAR為何採用彩色液晶螢幕時，以開玩笑的語氣說了句「不知道這是不是為時過早呢」，也許事實真的是如此吧，一直要到1998年，任天堂才終於推出使用反射式TFT液晶彩色螢幕的攜帶型遊樂器GAMEBOY COLOR，而且GAMEBOY COLOR的畫面表現能力在大多數情況下還遜於GAME GEAR，只能說SEGA這次還是如同往常一樣，走得太前面了一點。

嚴格來說，考量攜帶便利性與續航力等因素後，GAME GEAR本身真的不能夠算是台設計得成功的攜帶型遊樂器，但對於經歷過那個年代的玩家來說，那種彩色螢幕的震撼，可不是很容易就能夠忘懷，而GAME GEAR所推出過的遊戲中，也不乏時至今日依然讓人投入的佳作，如果您手邊還留著一台GAME GEAR，不妨拿出來重新感受那厚實的手感，回味它曾帶給您的感動。
