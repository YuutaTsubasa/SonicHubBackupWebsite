Windows 7 不是 Windows 7.0

👤 opus 🕐 2008-10-16 13:04:06

套一句老外愛用的網路語：WTF？Windows Vista Blog的 Mike Nash 解釋為什麼 Windows 7 要取這個名字，前半段和我們大家都知道的一樣：Windows 1、2、3，Windows9X 全部合稱 Windows 4、Windows 2000 是 Windows 5.0、Windows XP 是 5.1、WindowsVista 是 6.0。所以下一版的 Windows 應該是 7.0 版，取名為 Windows 7，一切都很美好對吧？

錯。因為 Windows 7 的版本號是 Windows 6.1。

官方的解釋是「改變基本版本號會造成第三方軟體相容性問題」，因此雖然「Windows 7 是個客戶端作業系統的重大進步」，微軟依然決定保持數字 6 的基本號，以確保客戶的軟體相容性。

不是我要說，如果相容性這麼重要，為什麼不把 Windows Vista 的版本號設定為 Windows 5.3？或是，在 Windows 7裡做一些設定讓它面對不聽話軟體時自動進入相容性模式不就好了？Windows 6.1...這樣的版本號很難讓人相信這是個「重大進步」呢。
