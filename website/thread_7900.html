(故事開始)

(Sonic HUB廣場 公寓住宅區 約晚上10:00)

即將進入深夜時段的HUB,街道上的人慢慢的變少了.許多的商家都要打烊回到住屋處休息.然而這個時候,Rain卻靠在Tesla Model S P90D的上面,感覺絲毫沒有任何睡意.
「....就在今晚嗎?何時會打給我呢?」Rain一副焦急的樣子.他緊握著手機,手不斷的輕微晃動著.莫約30秒後,他的手機響了.他馬上接起:「喂?」手機內傳來了聲音:「你好,這邊是Tesla的工作人員,我們約1小時候就會來到您的基地前,將您的Tesla Model S做100kWh的電池與馬達的升級.現在過去是否方便?」Rain則是邊走上Tesla的駕駛座,邊回道:「啊,當然方便.沒問題的,請快過來吧.我已迫不及待,要來體驗你們那最厲害的P100D了!」手機再度傳來聲音:
「好的,那麼沒意外的話,我們預計將於11點到達您的基地.」Rain嘴角上揚,笑了一聲,道:「好的,到時就麻煩了,謝謝啦!」接著,他將電話切斷,然後啟動Tesla Model S P90D的馬達,將檔位打入D檔,以傳送儀做為媒介,離開了HUB的世界.

Rain回喊:「稍等一下,快好了.」然後,B員工將視線轉回Rain上,他有點疑惑的道:「只怕...吵到您睡眠.」Rain搖搖頭:「不,沒問題的.至少我不會起床罵你好嗎?那就這樣說定了,記得把我挖起,謝了.」語畢,Rain關上基地大門,從一旁的小門走了進去,並將門隨手帶上.B員工緩緩走上卡車,並向一旁的司機C員工道:「你一定不會相信我剛才聽到他說什麼....」接著,卡車緩緩駛離Rain的基地門前,Rain也走進屋內,稍微把身體清洗一下之後,就寢.

已經做完電池升級的Tesla Model S [color=Red]P\"100\"D,緩緩停靠在基地的門前.而車上的駕駛就是剛才的B員工.他走下車,擔憂的道:「真的沒問題嗎?」,隨後,他便按了好幾下的電鈴.而在床上熟睡的Rain,當然是被吵醒了.「嘖...凌晨4點..不是吧...呃...喔喂!!」他原本模模糊糊的猜想是誰在這種時候造門,但他忽然想起:他的Tesla已經做好了P100D的升級,正在外面等著他.於是他匆匆忙忙,趕緊從床上跳起,直奔屋外的基地大門.過了約30秒,他氣喘吁吁的用力推開基地大門旁的小門,然後說:「唉呀~久等啦~」只是這舉動,完全是讓一旁的B員工給嚇傻了:「哇,你還真的...」Rain看向他:「哼哼,沒問題的啦.我不會撞毀的.萬分感謝你啦,錢已經付過了.那麼...我出發了!!」.隨即,他坐上了Tesla Model S P100D的駕駛座,緩緩的駛離基地.片刻,Rain啟動超時空傳送儀,回到了HUB廣場.

(過了20分鐘... SONIC HUB廣場 凌晨4:30 )

在HUB廣場幾公里外的道路,Tesla Model S P100D正閃著雙黃燈停在那邊.一段相當長的直線,讓Rain忍不住Hold住螢幕上的Ludicrous Mode紐.幾秒鐘,大螢幕從原本的資訊顯示形成了正處於飛梭狀態的流星,象徵的就是高速.緊接著....

「嘖嘖,還是老樣子250....不過0~100的加速居然可以到2.5秒,比P90D整整快了0.3秒,G力也從1.2變成1.4...哇,這推估,應該也有800多HP吧?真是太狂了啊這台.這世上還有哪台大型房車敢跟他比?有啦...唯一的一台...世界上最快的究極大型轎跑,Jet所御用的那一部Brabus Rocket 900 DESERT GOLD Edition...!!!」
唰~的一聲,Tesla在夜晚的道路中全速的奔馳著.5分鐘後,他回到了HUB廣場.街道上因是深夜的緣故,所以宛如空城般的安靜.Rain這時心裡想: 「如果我現在開的是LaFerrari的話,我應該會被所有住戶圍毆吧?不得不說,騜*的那V12油電混合引擎中的電,只是屬於\"增壓型\",讓他的瞬間爆發動力更強,而非純電行駛...唉,這樣子即使是用40Km/h的速度行駛,引擎依舊是開的啊.」Rain搖搖頭,同時加足電門,將Tesla開到公寓住宅區的門口,停在Rimac Concept S的旁邊,將電源關閉,下車.Rain看著外牆微微亮著黃色LED燈的高級公寓住宅區,喃喃的道:「小妹跟姊姊不是夜貓子,看到我留的訊息之後,應該都自己先睡了吧?我這樣進去應該是不會嚇到他們.」Rain走進了公寓住宅區的其中一棟,找到了自己的所住的房間.他輕輕的打開門,發現裡面果然漆黑一片.他輕聲的走向房間,用力一看.Sakura跟Venus都蓋著棉被,躺在床上熟睡著.Rain微微一笑,走到了Sakura的旁邊,輕輕摸了一下她的頭後,便躺在她的旁邊.
「小妹...妳還是那麼可愛哪~」Rain再躺下的瞬間,一股疲憊感忽然湧上,使他馬上就進入了熟睡狀態.
約5小時過後,時間來到早上9點半多.朝陽已經升起,街道上也慢慢的出現人潮.在公寓內,Sakura最先被刺眼的陽光給亮醒.「嗚嗯嗯...?」她揉了揉雙眼,接著發現旁邊有人躺在那邊,她用力一看,赫然發現是Rain,因此她嚇得直呼:「呀~Rain?!你..什麼時候回來的呀?」「喔呃啊?!」Rain被突如其來的聲音嚇到驚醒,並摔到床下.他趕緊爬起身,拍了拍身體,對Sakura直罵:「呃啊,小妹妳幹什麼啊,我只是提前幾小時回來這邊睡你何必大驚小怪啊?真的是...吵死了,一大清早的,就不能讓我好好睡嗎?」Sakura看到Rain如此氣憤,耳朵跟尾巴都垂了下來,低頭跪坐在床上道:「咿..嗯...對...不起嘛...人家只是...」而在一旁還好好睡著的Venus,耳朵動了好幾下,接著輕聲的道:「嗯嗯...怎..麼啦?喝啊啊」Venus打了聲哈欠,張開眼睛,發現Rain就站在她的正前方,與她四目相接.Rain率先回應:「呃啊...抱歉啊姊姊,是小妹太吵了對吧?」Venus先是斜眼看了一下Sakura之後,接著道:「嗯..沒關係啦.小妹又不是故意的,你昨晚留了字條說要回基地一趟,怎麼了嗎?」接著,Venus從床上站起,移動到Sakura的旁邊坐下,摸了摸Sakura的頭,給予安撫.Rain小聲的說:「就...去升級車子啦....」「嗯?升級車子?哪一台呀?」Venus轉向Rain問.Rain回應:「Tesla,就升級一些他們最新的配備那些啦....」而就在Rain說完話的瞬間,他的電話再度響起,Rain一看螢幕,疑惑的想:「怪事,才快9點快半而已,誰打來的?」.......

(經過一番協調後...)

「小妹,我有急事要借走妳的Rimac Concept S,可以嗎?一下子就好~」Rain掛斷電話,向Sakura問.Sakura則是毫不猶豫的答應:「當然沒問題,這邊~接住!!」Sakura將鑰匙丟給了Rain,他精準的接下之後,隨即衝上Rimac Concept S,然後開離.可就在開離的一瞬間...Sakura突然想起好像有事情沒有跟Rain說,於是就衝了過去.
「疑..我總覺得...呃..哇啊!!小紫,等等,人家有事...」就在她喊到一半時,已經來不及了.Rimac Concept S已經駛離了她的視線範圍.
唰~的一聲,Rain開著Sakura的那一部Rimac Concept S離開了HUB廣場,並往昨天晚上所載人的那個鎮上前進.大約經過了約15分鐘後,Rain開著寂靜之雷-Rimac Concept S到達了指定的地點,果然看見幾位熟悉的身影等著他.

「...?」Rain發現,Shackle的表情滿是不耐煩的樣子,還一副念念有詞.Rain嘆了一口氣,道:「嗯嗯...送你一張去程票和回程票,你應該要感到很滿足才是哪~而且再加上...我這一部號稱寂靜之雷的電能巨獸,可是比你上一次坐的那台Tesla轎跑還要快哦~」Shackle聞之,他睜大了眼睛,疑惑的看著Rain.

「應該是這實力嚇得說不出話了吧?呼~說實在的,這台可是真的很強呀...唯一可以把Tails軍團長的瑞典神獸ONE : 1逼得很死的一台啊~」
