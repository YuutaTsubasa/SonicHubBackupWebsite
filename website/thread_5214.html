以下正題
Dreams Come True（美夢成真）
是由吉田美和中村正人所組成的二重唱
成軍至今超過20年，唱片銷量超過4300萬張
是日本的長青流行歌手
維基資料
他們除了在流行樂壇持續發光發熱外，也參與了一些遊戲的音樂製作
其中當然包括Sonic系列
以下則是他們的一些歌曲，當中隱藏了Sonic遊戲的音樂
分享給大家：

Sweet Sweet Sweet

應該很多人聽過
因為在2006年的Sonic the Hedgehog中，這首歌與R&B歌手Akon合作
被翻譯成英文以Sweet Dream \06 AKON Mix 的姿態出現在Sonic篇的Credit中
不過其實早在1992年就已在專輯The Swinging Star中發行了，也就是這裡聽到的原版
大家應該都知道這是Sonic 2 Ending的變奏

MARRY ME?

要仔細一點聽才聽得出來，這是Sonic 1的Green Hill Zone變奏
這首歌創作的年代比較晚，收錄於1997年的SING OR DIE專輯中

薬指の決心（無名指的決心）

收錄於1991年MILLION KISSES專輯中
很明顯的，其中有著Sonic 1 Star Light Zone的旋律
不過副歌部分可能比較不明顯就是了
（PS：如果有人問為什麼是無名指的話，因為習慣上結婚戒指是戴在無名指上）, 123.192.110.145, 0, 0, 1, 0, 0, -1, 0, 0, 20, 200, 0
