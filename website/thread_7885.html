Amy嚇的跌坐在地板上：「影純焱，你怎麼還在這裡？」鏡子裡的人冷笑著說：「凡事總得有些代價，不是嗎？」Amy落寞地抱著頭：「不可能，不可能！」

Amy開始發瘋似的滿屋子亂跑，不管是撞牆，還是拿槌子捶自己，身子仍是愈來愈累，最後，她看著桌上的刀子，心一橫，往自己的咽喉狠狠的刺下去。

可身子卻不聽使喚的停住。

「想死，門都沒有！只要你活在這世上一天，就只有無止境的痛苦，明白嗎？」

( 門外 )

Amy發瘋似的衝出門，氣脫委頓，猶如將油枯燈盡的人一般，冥冥之中，好像有個身影往自己走來。「道長，你快救救我吧！」

( 施法陣壇上 )

劉斌喃喃的念著咒，只見四周風雲變色，大風狂沙四起，劉斌突然眼睛一瞪，手望Amy指了過去：「現！」只見一個黑影從Amy身後出現：「好小子，竟能把我從凡人身上打出。」突然一掌拍去，劉斌一劍擋下，那黑影就四散而去。

劉斌拿起手上從影純焱拿來的檔案：「可惡，被騙了！」隨手一丟，Amy看著白色的檔案資料上寫著：「甦魂計畫。」四個大字。

劉斌冷冷地看著Amy：「下一次，不要再亂召喚了。」


自創角色介紹

魂越派，天下神秘門派之首，天下無幾人所知，只知其中最有名的「五素長老」，及「魂越七才」。以無情，無感，無念和敵人展開殊死暗戰。

姓名：劉斌
魂越掌門，武功幾乎天下第一，練就一身「魔絕寶法」以絕情，右半面有舊疾，帶一青銅面具遮掩。自稱御劍第一，畫符第二，下棋第三。曾和五素長老因影純焱喜歡Shope這位小女孩一事破壞門規，耗七天七夜一同封印影純焱，封印後遭懷疑而趕出門派，發瘋一年後回歸，成為魂越掌門。

成為掌門後嚴格禁止「有情」和「多情」，使自己成為一位無情之人。

武功：御劍術，軒轅劍法，七劍歸一訣
法術：八方歸元術


姓名：林寒
是林家「冰鳳穿魂」的傳承，練就一套徒手煉針的訣法，身上有一套「三十三套暗器」，是天下第一到三十三名的暗器全套，醫術高明，書法也是一絕。

武功：劍氣不寒，冰鳳穿魂針


姓名：楚風雲
自練「三昧真火」及「焚心訣」，觸及之物皆可燃燒，曾和Blaze決鬥過，最後瀕臨生死存亡之際，以一己貫之神功險勝

武功：風雲劍法


姓名：許處泉
身法高絕，號稱「在快一點點就和Sonic一樣快」，好彈琴，自創「刀光劍影鬼哭神號訣」，聲如刀劍，聞者即死。

武功：刀光劍影訣


姓名：上官安
知人所不常知，話說他常坐輪椅 ( 據他的說法，是行動式的轎子。 )，是因為不願意向凡夫俗子一樣走路。自承上官家知名的「空虛劍法」。

武功：空虛劍法


姓名：西門昕
常以扇子作為武器，事實上奪人性命卻以一把絕世寶劍為主，無人知道那把寶劍是甚麼，只因為連劍都看不到就死於劍下。

武功：一劍西來，劍神一笑，五陣凌風訣


姓名：皇甫中
七才中年紀最小，卻是唯一一位能成功練完武林最致命的「鐵翼神衣」的人。

武功：鐵翼神衣

本人剛好看畢閣下之作品，發覺閣下看來是極喜歡武俠小說之類的世界，亦希望將此套入Sonic世界觀其中。首先這當然沒問題，就算Sonic電玩作品也有進入中世紀亞瑟王及一千零一夜之故事內。事實本人也很久之前曾經寫過Sonic進入三國時代之同人文，但這就不贅。我就唯有在此記下一些本人的觀後感及一些建議。

其實看畢後，除了故事偏短(但閣下有解釋)外，整體就覺得有點怪，就是根本將古代武俠場景完全套用在內，連Sonic們都變得…根本不像一向給我們很有現美式風調、連思考模式作風現代化的Sonic一眾角色。本人並不反對閣下對武俠的熱愛。然而結果是，給一些Sonic愛好者們看起來，他們根本就走了樣，完全變了另一個角色。一個不時會說美式俗話，不時吐幾句英文的Sonic一眾，是很難說話突然會變得如古代文人般文皺皺的 -- 尤其是一個喜歡遵從自己風格，愛自由奔放的Sonic。

既然閣下所寫的是Sonic同人文章，就請先詳細瞭解Sonic們的角色特點，日常生活是怎樣，與他人相處時的言行習慣會是啥，甚至他們會以甚麼方法去思考。想說是，他們混進截然不同的世界不是問題，但不可能連性格也被大改。當然本人從前在寫三國同人文時，也曾有位寫小說極佳的會員意見予本人，認為Sonic的說話方式帶有中國古代戲一樣的臺詞需要修正。到往後本人都將盡其量把原本來自未來的Sonic之語句變得現在代日常的白話說話方式，到現在看來這實在正確。

再者，如閣下所說，是自己的武俠角色進入Sonic的世界，既然如此的話…正如上篇回應所言，總會有點現代科技產物吧!須知sonic的世界觀是一個怎樣的景物?正是一個現代甚至近未來的世界，是有各種各樣的科技電器，外景也是不少高聳入雲的高樓大廈，有車有飛機有鐵路的。然而閣下作品出來的場景就是只有正是如武俠小說內的高山與一些簡單的亭樓。就算現代的深山野嶺，也不時會有些攜著現代裝備的探險隊作科學探索吧!不可能只有一部手機，還要未見有登場過。

還有Sonic們終究也屬現代人，問對衝突，解決問題的手法總會要用上現代日常手法，而不是滿有招式般的刀劍比劃；總會有警察甚至一些GUN(Sonic世界內的正規軍隊)會介入一下的。在發現一些武林秘笈時，也總會召喚Tails來用儀器掃描一下作分析，亦會用現代的術語解釋箇中原理嘛。要記住，正如閣下所言，他們現正身處現代!除非是相反Sonic們進了古代武俠世界，但就算是古代武俠世界，Sonic難不會用些他們一貫如電玩內的方式及思想去面對新奇的事物及未知的命運嗎?

結論就是整個故事過於武俠，搞得他們不恊調。既然閣下喜愛武俠小說，也都想說一下金庸這位人所共知的人物。金庸寫武俠小說時，並非只於運用古書古籍去推想故事；反之汲取了西方文學的營養，作品中還參考一些古希臘悲劇、莎士比亞名著之類：亦會用舞台戲劇，電影的方式來描寫情節。再利用現代的思想及一些政治事件的抒發用在武俠小說內。比如「射鵰英雄傳」、「神雕俠侶」就是受到「羅密歐與朱麗葉」及「哈姆雷特」等莎士比亞名著所影響；而寫「笑傲江湖」時，正值文化大革命，金庸寫此正是有感而發。

但就算如此，也可以作更進一步的。其實兩個世界不同的人與事衝擊一起，也是好的題材來。既然古代武俠人物來到現代的Sonic世界，當然有很多事物也是未知!就此可以把武俠人物來到近未來的世界會有怎樣的衝擊?他們如何理解習慣及不同的文化差異，甚至去適應。再者，Sonic一眾又如何用他們已佑的方式去協助這群俠士，令他們驗到不一樣的戰鬥風格。這些到都是可以將之發揮的題材。這當然也有電影可作參考，這就是1989年的香港電影「急凍奇俠」，內裡的從古代而來的主角也挺有武俠風格，又有拍攝主角怎樣面對現代城市的一切。可以給閣下往後發展有些幫助。

當然，閣下也要繼續認識Sonic一眾吧，既然稱得同人小說，這必要花下心機。還有就要閱讀一些武俠小說以外的書籍，認識不同的事物也有助寫作。同時間，平常也要留意時事新聞，多閱報，理解現實各事件的事態發展；亦要主動接觸不同的知識及事件，此舉對編寫故事時就更得心應手。

但當然，若果覺得這會極不符合自身風格，不妨考慮直接寫原創武俠小說會更好。再當然這是本人愚見，以上。
