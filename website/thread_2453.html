怪盜暗影(4)

👤 jessie 🕐 2008-11-16 06:54:42
[size=14.0pt]怪盜暗影[size=14.0pt](4)[/size][/size]
[size=14.0pt] [/size]
[size=14.0pt] [size=14.0pt]我在[/size][size=14.0pt]Rick[/size][size=14.0pt]的[/size]府第門前按下門鈴，「來了[font=Times New Roman]![/font]」一個白髮老翁走出來應門，他向我打量一下，便問我[font=Times New Roman]:[/font]「我是這裡的管家，請問你是誰呢[font=Times New Roman]?[/font]」我胸有成竹，早有準備[font=Times New Roman]:[/font]「我是已經向[font=Times New Roman]Rick[/font]預約好的水電工程師，來為你們檢查和保養電錶箱的。」「[font=Times New Roman] [/font]哦[font=Times New Roman]...[/font]是嗎[font=Times New Roman]?[/font]」老管家把我領到屋子的電錶箱前「這邊，請[font=Times New Roman]![/font]」「哦，是這個啊[font=Times New Roman]?[/font]」我打量著電錶箱說。老管家說[font=Times New Roman]:[/font]「如果沒有別的事，那我去忙別的了。」說完便轉身離去。[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font][/size]
我拿出一個鉗子和在那位水電工程師身上偷回來的電錶箱圖示，我看了看那張圖，正當我想把連接到「世紀之光」展覽廳防盜系統的電線剪掉時，以為這樣做，防盜系統便不能如常運作了。可是，我差點兒忽略了一個細節[font=Times New Roman]:[/font]原來這個電錶箱是用內電的，即是說就算把電源切斷，電錶箱內部還是會有電源供應的，防盜系統一樣會照常運作的。似乎我的估計錯誤了，只好另行再想方法了，不過無論如何，還是趕緊進屋再說吧。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
我小心翼翼打開窗門，然後手抓窗邊，身子向前作出一個三百六十度的淩空翻騰，最後輕盈著地，我只用了不到兩秒的時間便進入屋內。我正在觀察我身處的位置，只看見一個個堆得高高的紙皮箱，地板佈滿塵埃，還有一堆堆的舊書，看來這應該是雜物室吧。不過這裡的灰塵實在令我的鼻子非常難受，還是儘快離開這裡吧。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman] [/font]我輕輕地扭開門把，門外便是大廳了。正當我在思考下一步的行動，突然，我聽到一些由遠至近傳來的腳步聲，我靜心傾聽，唔[font=Times New Roman]...[/font]這似乎是老管家的腳步聲。「糟糕了[font=Times New Roman]![/font]」我心中暗叫不妙「萬一被老管家發現我在這裡而不是在檢查電錶箱，那他不就懷疑起我了嗎[font=Times New Roman]?[/font]」難道老天爺真的不保佑我嗎[font=Times New Roman]?[/font]他已經看到我了，還向我大叫[font=Times New Roman]:[/font]「咦[font=Times New Roman]?[/font]你為什麼會在這裡呢[font=Times New Roman]?[/font]你不是應該在檢查電錶箱的嗎[font=Times New Roman]?[/font]」我現在的處境真是糟透了[font=Times New Roman]![/font]我該怎麼辦呢[font=Times New Roman]?[/font]
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman] [/font]我十分緊張地思考對策，額角滲出冷汗，可能老管家發現到我的臉色非常難看，所以他問[font=Times New Roman]:[/font]「先生，你哪兒不舒服了[font=Times New Roman]?[/font]你為何在不停冒汗啊[font=Times New Roman]?[/font]還有，你的臉色很蒼白啊[font=Times New Roman]![/font]」我仍然緊張如故，堅守著「沉默是金」的金句，一言不發，臉色變得愈來愈白，直至到老管家問我[font=Times New Roman]:[/font]「你需要一些葯物嗎[font=Times New Roman]?[/font]」就在這同時，我的腦海中閃過一絲靈感，是以我馬上說[font=Times New Roman]:[/font]「咳咳[font=Times New Roman]...[/font]我沒事，只是天氣太熱罷了，其實我是特地來找你的，因為我有些很重要的事一定要通知你，只是不停地找也找不到你，所以有點緊張而已。」
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
「什麼[font=Times New Roman]?[/font]」管家似乎受寵若驚，他睜大眼睛說[font=Times New Roman]:[/font]「請問，你[font=Times New Roman]...[/font]你有何貴幹呢[font=Times New Roman]?[/font]」[font=Times New Roman]<<[/font]待續[font=Times New Roman]>>[/font]
