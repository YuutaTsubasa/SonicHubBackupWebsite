GAMEの「Dreamcast」逆襲？

👤 aoifh 🕐 2009-09-17 01:15:38
現今次世代主機平台需要哪些要素？
Dreamcast由日本老牌遊戲製造公司GAME所改造
歷經10年從初代主機在現在即將出世的Dreamcast
家庭遊樂器的老玩家都知道
GAME是怎樣從領導者到完全退出遊戲主機開發
但是GAME亡靈即將再現
主機規格:
OS︰Vista64
CPU︰Core 2 Duo E8400 3.0GHz 或更高 已知Q9550 OK
RAM︰DDR2-2GB x2 [4GB]
SSD︰A-DATA 64GB
M/B︰ZOTAC GeForce 9300-ITX WiFi
GRAPHIC︰NVIDIA GeForce 9300 mGPU
ODD︰Panasonic UJ-120 [BD-ROM / DVD±RW-RAM]

まさにセガの亡霊、ドリームキャストをCore2Duo 3GHz/メモリ4GB/SSD64GB/BDドライブ搭載PCに改造してPS2を動作させる衝撃ムービー


「セガのハード事業復活を祈って最終兵器Dreamcastを大改造して宿敵ハードを体得させてみました」というムービーがすさまじい内容となっています。前編9分33秒・後編7分52秒というボリュームであることからもわかると思いますが、趣味の領域を超えており、まさに「プロの犯行」「野生のプロ」と呼ぶにふさわしいとんでもないレベル。細部の完成度もすごいのですが、ムービー自体のレベルも高く、ものすごくわかりやすい3Dモデルを多用し、電子音声ガイド付きになっているため、ちゃんと聞いていれば何をしているのかは大体わかります。

逆襲のセガと言っても過言ではないムービーの再生は以下から。


ドリームキャストをCore2Duo 3GHz搭載PCに改造し、PS2を動作させてみた (前半)‐ニコニコ動画(ββ)
ドリームキャストをCore2Duo 3GHz搭載PCに改造し、PS2を動作させてみた (後半)‐ニコニコ動画(ββ)

ニコニコ動画のアカウントがない人用に、YouTubeバージョンは以下にあります。
YouTube - Dreamcast in Core2Duo 3GHz running PS2 [1/2]


YouTube - Dreamcast in Core2Duo 3GHz running PS2 [2/2]


1本目の出だしからして半端ないことになっており、「皆さんは、覚えていますか？苦しくても自前ハードで奮戦していたセガの歴史を……」というナレーションが入り、つかみとしてあの湯川専務による自虐CMを持ってくるあたりからして、往年のセガファンは涙が止まらないはず。


そこからはなぜかBGMがトレジャーの傑作STG「斑鳩」になり、いよいよ本編スタート。「CMに涙して、5台も本体を買ったあの日から、10年も経ってしまいました」という驚愕の告白。


そして、「ハード事業撤退の屈辱をかみしめて、この小型ハードを高性能PCとして復活させました」という言葉とともに、怒濤のごとく改造が始まります。



作者の解説によると、2001年に一度製作しているとのことなのですが、それにしてもこの執念は尋常ではありません。


今回は新たに3つの目標を設定したとしており、「宿敵ハードを動作させるためにTDP(設計上想定されるマイクロプロセッサの最大放熱量)65W程度のCPUを冷却可能な構造とし、Core2Duoだけでなく、Core2Quadも視野に入れて設計する」


「実機部品を含め必要な部品すべてをCADでモデリングし、設計する」


「ケースへの部品接着を原則禁止とし、組み立てと分解を行えるようにネジ止め構造で設計する。軟化温度が低いホットボンドの使用は論外なので厳禁とする」


また、CPUとして低消費電力がウリであるIntel製CPU「ATOM」を使わない理由についても解説しており、「低消費電力ですがクロック当たりの性能が極端に悪く、Core2Duo3GHzの20％以下の性能しか持ち合わせていません。デスクトップに低性能な低消費電力CPUを搭載するのは妥協以外の何者でもない」と切って捨てています。


ここからが本番。中身はかなり高密度であるとして、下から冷却部、USB基盤部、マザーボード、無線LANモジュール、SSD、Blu-rayドライブ基盤、ドライブ駆動部となっています……というような感じで解説が始まり、制作順に細部の紹介が始まります。


冷却ファンの選び方、実機テスト、テスト結果から導き出されるベストな方法の採用……というようにしてこのあとは延々と続くのですが、見ているだけでも「これが……本当の自作PC！！」と感激できるクオリティです。
