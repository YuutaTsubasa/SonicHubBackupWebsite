比爾蓋茲：Window 7 明年推出

👤 opus 🕐 2008-04-06 04:21:03


昨天微軟說道願意為 eeePC 將 Windows XP 延長壽命至 2010 六月或者是 Windows 7 推出之時。如果微軟真的計畫在 2010 年推出 Windows 7，我們可以這樣推測--Windows 7 愈早推出代表愈快結束 Windows XP 的壽命。

今天比爾的一席話似乎證明我們的推論，比爾蓋茲告訴投資者 Windows 7 可能在「明年推出」，而且言談中近乎「滿滿的愛。」

不過據我們所知，Windows 7 的推出時程並沒改變，比爾大叔說得可能是 Beta 版本的推出，還是 Windows 7 的研發有著超乎預期的進展呢？這就是不能說的秘密了，不過對一般電腦使用者來說，VISTA 還算是相當新穎的名詞阿∼WOW∼
