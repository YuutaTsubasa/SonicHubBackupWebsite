用Game Maker設計這個遊戲, 特點是與其他的Sonic 2D遊戲與別不同.
------------------------------------------------------------------------------------------
第一次操作 (已經可以變更！)
F1: Game Information
F4: 全螢幕
F9: 複制圖片
Enter: 暫停 (重新開始/幫助/操作)
I: 前行
U: 左Quick Step
O: 右Quick Step
J: 左行
L: 右行
K: 滑行
空白鍵: 跳
Z: Sonic Boost/選擇
X: Light Speed Dash/取消

------------------------------------------------------------------------------------------
現在Sonic Quick Step Challenge已經支援PS3, XBOX360手制
首先下載MotioninJoy Gamepad tool
http://www.gamefront.com/how-to-use-a-ps3-controller-on-a-pc/
之後看以下影片
http://www.youtube.com/watch?v=cvVDW3cCRYE

必須要把Button改成以下一樣和每次開DS3 Tools也要按Save Changes
Button 1: Triangle/Y
Button 2: Circle/B
Button 3: Cross/A
Button 4: Square/X
Button 5: L2/LT
Button 6: R2/RT
Button 7: L1/LB
Button 8: R1/RB
Button 9: Select
Button 10: Start
Axis x: Left stick x
Axis y: Left stick y
