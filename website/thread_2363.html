[font=Times New Roman][/font]
[font=Times New Roman] [/font]二人低頭沉思良久，[font=Times New Roman]Silver[/font]首先打破沉默[font=Times New Roman]:[/font]「在沒有足夠證據之前，我們最好不要輕舉妄動。」[font=Times New Roman]Blaze[/font]點頭說[font=Times New Roman]:[/font]「嗯，我們還是趕快到皇宮去看看那個[font=Times New Roman]Kidult[/font]在幹什麼鬼吧[font=Times New Roman]?[/font]」兩人一起向地下圖書館的出口走去。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman]
Silver[/font]把地下圖書館出口的門打開，可是一開門，二人立即用力捂著鼻子「哇[font=Times New Roman]![/font]」[font=Times New Roman]Silver[/font]痛苦地慘叫「為什麼會這麼臭的啊[font=Times New Roman]?[/font]」「噢，抱歉」[font=Times New Roman]Blaze[/font]苦笑說「我忘了告訴你，這條通道是通往皇宮的地下水渠，只有這條通道是最隱閉的了。」[font=Times New Roman]Silver[/font]問[font=Times New Roman]:[/font]「那是不是意味著我們一定要走這條通道了吧[font=Times New Roman]?[/font]」「對啊[font=Times New Roman]![/font]」[font=Times New Roman]Blaze[/font]理所當然地回答。[font=Times New Roman]Silver[/font]聽完之後，差點兒昏過去，但為了不在[font=Times New Roman]Blaze[/font]面前表現出懦弱的一面，所以他決定支持下去，不讓自己昏倒。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
「來吧[font=Times New Roman]![/font]」[font=Times New Roman]Silver[/font]自告奮勇地走前頭，並拉著[font=Times New Roman]Blaze[/font]的手「我們一起向皇宮出發吧[font=Times New Roman]![/font]」說完便自顧自地走向前方「喂[font=Times New Roman]!Silver[/font]，小心啊[font=Times New Roman]![/font]」[font=Times New Roman]Blaze[/font]大叫「你前面有一個水窪，別踏下去啊[font=Times New Roman]![/font]」可惜晚了一步，[font=Times New Roman]Silver[/font]一腳踏到那個水窪上，然後滑倒，污水淺到[font=Times New Roman]Silver[/font]身上，讓[font=Times New Roman]Silver[/font]的白毛變成了黑毛。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman] [/font]「[font=Times New Roman]Oh! My God![/font]」[font=Times New Roman]Silver[/font]再次慘叫起來「我的白毛啊[font=Times New Roman]![/font]我不要變黑[font=Times New Roman]![/font]」「冷靜點兒[font=Times New Roman]![/font]」[font=Times New Roman]Blaze[/font]連忙安撫他，[font=Times New Roman]Silver[/font]快要哭起來了[font=Times New Roman]:[/font]「可是，我這副模樣怎樣見人啊[font=Times New Roman]?[/font]」「不是啊[font=Times New Roman]![/font]我覺得你現在還滿好看的啊[font=Times New Roman]![/font]」[font=Times New Roman]Blaze[/font]笑著說，突然，她的腦海閃過一絲念頭「[font=Times New Roman]Silver[/font]，你等我一會兒，我去圖書館拿些東西。」說完便轉身回到圖書館的歷史文獻書架上，拿了一本記載著十年前發生的大事的文獻。
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman]Blaze[/font]把它翻到其中一頁，對[font=Times New Roman]Silver[/font]說[font=Times New Roman]:[/font]「看[font=Times New Roman]![/font]你認識這個人嗎[font=Times New Roman]?[/font]」「誰啊[font=Times New Roman]?[/font]」[font=Times New Roman]Silver[/font]集中目光去看文獻上的這個少年，只見他長著一頭黑髮，在俊朗的臉上長著一對目光冷酷的眼睛，眼神無情中帶一點感性，烏黑又修長的四肢配上紅色，經過改裝的超級運動鞋，令人一眼望去就知道他是一個速度性的人物「咦[font=Times New Roman]?[/font]」。[font=Times New Roman]Silver[/font]好奇問道「這個是誰啊[font=Times New Roman]?[/font]」
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman]Blaze[/font]唸出文獻上的資料[font=Times New Roman]:[/font]「這個少年叫[font=Times New Roman]Shadow the Hedgehog[/font]，他和一個叫[font=Times New Roman]Dr. Eggman[/font]的瘋狂自戀科學家在十年前進行了一個關於七個[font=Times New Roman]Chaos Emerald[/font]的實驗，這個實驗關於探討[font=Times New Roman]Chaos Emerald[/font]的力量可以去到的程度，當時[font=Times New Roman]Shadow[/font]和[font=Times New Roman]Dr.Eggman[/font]是一對情同手足的好友，所以[font=Times New Roman]Shadow[/font]毫不猶豫地就當了[font=Times New Roman]Dr.Egggman[/font]的實驗品，可是實驗最終失敗，[font=Times New Roman]Shadow[/font]就昏迷不醒。但在某一天，[font=Times New Roman]Shadow[/font]突然醒過來，並永遠離開了[font=Times New Roman]Dr.Eggman[/font]，自今下落不明，而[font=Times New Roman]Dr.Eggman[/font]卻無悔改之心，仍然繼續他征服世界的野心。」
[font=Times New Roman]------------------------------------------------------------------------------------------------------------[/font]
[font=Times New Roman] [/font]「但是」[font=Times New Roman]Silver[/font]大惑不解「你說這麼多歷史給我聽，究竟是想幹什麼啊[font=Times New Roman]?[/font]」「你還不明白嗎[font=Times New Roman]?[/font]」[font=Times New Roman]Blaze[/font]既緊張又興奮「我想到一個大好方法了[font=Times New Roman]![/font]」「是什麼方法啊[font=Times New Roman]?[/font]」[font=Times New Roman]Silver[/font]問道[font=Times New Roman]<<[/font]待續[font=Times New Roman]>>[/font][size=14pt][/size]
