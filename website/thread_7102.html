可以

開啟 ConfigurationTool
切換到 Input Configuration
Input Type 選 Keyboard

點圖中的按鈕，在按你對應的鍵
完成後儲存即可

上-數字鍵5
下-數字鍵2
左-數字鍵1
右-數字鍵3
X-X
Y-Z
A-A
B-S
LB-數字鍵4
RB-數字鍵6
