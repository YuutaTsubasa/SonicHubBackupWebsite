微軟：Windows 7 將不直接支援 SLI / CrossFire，因為...

👤 opus 🕐 2008-11-07 09:18:12


SLI / Crossfire 目前還是在高階的 Gaming PC 上比較常見，但未來將內建的顯示晶片和一張低階的外接式顯卡做 SLI /Crossfire 應該也是很多廠商會採用的方案，因此微軟會直接跳出來說 Windows 7 將不支援 SLI / CrossFire本身就是件蠻奇怪的事。但話雖這麼說，微軟確實是沒有支援 SLI / Crossfire 的義務，只要 Nvidia 和 ATI (AMD)有寫驅動程式就好，可是微軟不支援的理由就有些好玩了：

「（SLI / Crossfire）有可能會讓系統不穩定，造成不好的使用者經驗。」，同時微軟「強力不建議系統製造商採用此類系統」。

這是怎麼回事？SLI / Crossfire 有這麼不穩定嗎？
