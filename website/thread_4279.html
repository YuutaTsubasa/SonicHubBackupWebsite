👤 rescueTails 🕐 2009-10-22 14:05:03
全文翻譯:

SEGA的fans，發動引擎吧。因為這次在Wii上，你們不會再看見那個意大利水管工和他的一干伙計們開著車子橫衝直撞了，這次的明星是一隻藍色刺猬，一頭整天拿著沙球搖來晃去的猴子，一個穿著公雞服裝的男孩，以及玩家們前所未見的由第三方遊戲人物組成的龐大陣容，再加上瘋狂的賽道，勁爆的賽車，種類繁多的Power-up道具，這就是Sonic & SEGA All-star Racing（以下簡稱SSAR），這個立志要成為可以與蘑菇水管王國相抗衡的超級賽車遊戲。



Sumo Digital的開發者們首先想到的並不是該怎麼把一大堆自己喜歡的遊戲人物塞進遊戲中去，而是如何去製作一款穩定的、有趣的、街機式的賽車遊戲，並且保證遊戲的操作通俗易懂，拿起來就上手。 （例如使用Wii的遙控桿當方向盤，或者使用副搖桿Nunchuk的方向控制器；也可能會有傳統的操縱桿模式）

“這遊戲從一開始就是百分百的賽車遊戲，”製作人Steve Lvcett說道，“在我們之前，SEGA有那麼些優秀的賽車遊戲，例如Outrun，Daytona USA，Visual Racing，還有SEGA拉力賽，我們想要繼承它們的精神。所以這一次，我們更加追求將速度與漂移技巧結合在一起，讓大家體驗到駕駛的樂趣。在策劃過程中，SEGA建議我們應該嘗試將武器加入遊戲當中，我們也就照做了！剛開始我們自己也有些質疑這些想法——因為這樣不可避免地就會把我們與其它一些類似的遊戲進行比較，但是加入這些要素讓遊戲更具有可玩性與技巧性。這樣倒讓遊戲變得更好玩了，畢竟，這是遊戲製作時最關鍵的問題，不是嗎？”


夢幻陣容（Dream Cast）



對於這類游戲，不可或缺的元素就是那些深受玩家熱愛的人物，而SSAR正是要將這一點呈現給大家。如果你以前擁有過Sega Master，Genesis（MD），DC這類游戲機，那你這次可真的要升入藍色天堂了。正如標題所示，Sonic the Hedgehog以及他的一些夥伴（包括Tails，Amy，Big,Shadow，以及蛋頭博士）都會出現在選手席裡。而我們的花名冊裡總共還有20位以上的人物，包括AiAi（超級猴子球），Amigo（Samba de Amigo），Billy Hatcher（Billy Hatcher與巨蛋），Beat（塗鴉大街），Alex Kidd (Alex Kidd系列），Ulala（太空頻道5），以及芭月涼（莎木3）。

當然並不是什麼人都可以往這款遊戲裡鑽的（我們起先還期待戰斧裡的大反派蝰蛇騎著龍閃亮登場？），不過這個名單還是足以讓很多玩家滿意了。 Lycett坦誠：“挑選角色是個充滿了歡笑與心碎的過程。我們當然想滿足全天下人的口味，但是我們總得在十年內把這個遊戲做完吧！”

Sumo Digital之前就已經拿大量的SEGA角色做過電玩，例如SEGA Super Star Tennis（SEGA全明星網球），但是他們還是首先諮詢了日本的人物設計師，以確保所有人物登場的設定符合他們原來的樣子。 “我們特意去了日本，到訪了各個不同的部門。”Lycett說，“我們與所有人物的原設計者們密切合作，以確保讓玩家們看到他們喜歡人物的本來面貌。從模型、動畫、音樂到語音都回到日本總部全方位查透透，不斷地修飾雕琢。很多次他們還把車子的設計圖以及一些想法也發給我們……他們確實對此很用心，也很支持我們，能榮幸地與他們以及他們所創造出來的那些人物們合作，實在是件很棒的事情。”

每一位車手都會極力秀出自己的個性，他們不僅僅擁有屬於自己的動畫（可以看看他們領先或者落後而意圖趕上時的動作），還有自己的獨家絕門專屬賽車。比如，Sonic就開著一輛足以誇張地體現他速度的賽車，而Tails則開著他那眾人皆知的飛機，芭月涼騎的則是摩托機車，每個人物都有各自的優缺點，這主要取決於他們的座駕。跑車更適合堅硬的路面，但越野賽就不拿手；摩託在各種路面上速度都很一般，但加速度以及操縱性能都更好；越野車則通吃各種地形，例如草地，泥地，沙地和水面。飛行器沒有任何地形優勢，但可以在跳躍時突然加速。

每個人物都會使用“全明星”技能，這是他們各自的看家本領發揮出閃光的時刻。雖然你只能通過拾取地上的隨機道具來獲得使用全明星技能的機會（在你表現糟糕的時候更容易得到），但是它們卻是整個遊戲裡最酷，最炫，最強大的能力——就好像全明星網球里特殊動作一樣。例如Sonic，正如各位所預料的，他會變成黃金超級啃毛怪(大誤)，從此一統整個比賽；Amy會用錘子把所有擋道的傢伙錘到阿米吉特頓；AiAi會呼叫他的超級球夥伴們把對手敲成照片；Amigo則會令所有對手站成一排跳康恰恰。

還有其他很多強力道具，不過沒有前面那個厲害就是。比如Sonic的高速跑鞋可以讓你高速沖刺；彩球炸彈會阻礙對手的視線；防御用盾牌；一種名叫大號炸彈的道具，物如其名，可以把你的敵人炸得團團轉；以及混亂之星，據說這是Lycett最喜歡的道具。 “這玩藝打中你的話，整個畫面就會１８０度倒轉，”他解釋道，“你的腦子肯定不會那麼快地作出反應調整左右方向，儘管我們沒有對操控作任何手腳，但是這樣也足夠把你嚇到了。”當然，比起使用道具，駕駛技巧也是很重要的，通過漂移和跳躍（輕揮Wii搖桿），你可以獲得加速獎勵，讓你一舉超越對手。



超級車手（Mega Drive）

想要征服所有２０多條賽道，你必須使出渾身解數。自然這些賽道的設計靈感都來自經典的SEGA遊戲，包括Seaside Hill，Casino Park，Final Fortress（Sonic the Hedgehog），Blizzard Beach（Billy Hatcher），Tokyo-To（塗鴉大街），Curien\s Mansion（死亡之屋），以及超級猴子球和Samba de Amigo的場景。

千奇百怪的賽道包含了各類地形，以及跳躍平台、螺旋式轉彎、捷徑、還有與該賽道主題相切合的危險地段等要素。 “在Monkey Balls的主題賽道裡，你就必須躲避滿地亂滾的猴子球。”Lycett描述道，“在Seaside Hill，你會遇到早期Sonic遊戲裡的機器人。我們盡可能地加入更多的配角、場景以及敵人，讓整個環境能與你回憶裡的相符。”

同樣的懷舊理念也應用在了音樂上。作曲師１Richard Jacques——曾經多次為SEGA賽車遊戲諸如Sonic R等配樂的音樂師正在為遊戲創作新的音樂，而SEGA日本的諸多作曲師也加入其中。 “除此之外，”Lycett說，“我們還獲准到SEGA那龐大的音樂庫裡挑選出我們認為最棒的賽車音樂。在全明星網球裡我們有超過８０首音樂——這一次，我們有更多，更多，更更多的音樂奉送。”

除了原汁原味的SEGA風格之外，遊戲還包含有眾多模式，包括傳統的大獎賽模式，時間挑戰，以及多達５０個單人挑戰任務（例如在規定時間內擊倒一定數目的敵人，完美漂移，收集rings等）四人分屏遊戲（比賽與對戰模式），以及八人聯網對戰。

Sonic & SEGA All-star Racing擁有成為頂級卡丁式賽車遊戲的所有潛質：穩定的遊戲操控，偉大的人物，以及其他遊戲內容（包括一大堆可以解鎖的元素）。 Mario大叔得好好看看他的後視鏡了——Sonic和他的伙伴們隨時都準備著要把他的荷包搶空呢。, 125.84.232.114, 0, 0, 1, 0, -1, -1, 0, 0, 40, 240, 0
