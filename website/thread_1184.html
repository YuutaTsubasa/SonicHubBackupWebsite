使用方法: (By Wii 玩家俱樂部)
1. 先下載相關的檔案，並將它解壓縮。

2. 在你的主機上，把你的遊戲 Save 檔 copy 一份到 SD Card 上。
(每個遊戲有其特定的目錄，所以，要跟著其目錄結構放 Save 檔)

3. 用讀卡機將 SD Card 連上 PC，將下載來的 Save 檔 (data.bin) 蓋掉SD card特定的子目錄下的data，如火影的目錄就是 RNXJ。

4. 複製好後，將 SD Card 插回主機去。

5. 先把主機上該遊戲的 Save 紀錄刪除 (一定要做)

6. 然後再把 SD Card 上的遊戲 Save 檔複製一份到主機上。

大功告成，盡情享用吧!

下面再補充一下給新手的一些提示(反白):
1. 新手上路，你需要準備的東西有:
- Wii主機 (謎之聲: 廢話)
- 遊戲片 (謎之聲: 麥來亂!)
- SD card (謎之聲: 這個粉重要!)
- 讀卡機 (謎之聲: 出門在外必備良品!)
- PC (謎之聲: 沒有你我做不到!)
- 大腦 (謎之聲: 這太汙辱人了)

2. 搞不清楚SD卡上那個才是遊戲的存檔!?
遊戲的存檔是以Data.bin的檔案名稱存在遊戲對應的目錄下!!
例如:
$:\\private\\wii\\title\\****\\data.bin
符號$: 代表的是用讀卡機在PC上讀取你SD卡的磁碟機代號。
\"\\private\\wii\\title\\****\\\" 這一串是路徑，最後的****是指遊戲的目錄名稱，每個遊戲都有自己的名稱。
$:\\private\\wii\\title\\RRBJ\\data.bin 是雷曼兔的紀錄。
$:\\private\\wii\\title\\RBBJ\\data.bin 是轟炸超人的紀錄。
$:\\private\\wii\\title\\RNXJ\\data.bin 是火影忍者的紀錄。

3. 從網路下載來的記錄檔!?
一般都是經過壓縮的檔案，先解開後，有些包含了完整的路徑，有些只有data.bin的檔案。
不管路徑是否完整，重點是data.bin一定要有。
如果下載的是火影忍者的紀錄(data.bin)，就一定要去覆蓋或取代SD卡上RNXJ目錄下的data.bin才行，然後再copy回主機。

4. 主機上的紀錄一定要先刪除嗎!?
是的! 因為遊戲只會讀取主機上的記錄檔。如果你還想玩自己的紀錄，建議先將自己的紀錄備份到PC上，以備日後不時之需。

希望這幾個提示對新手能有些幫助，也希望老手們能多多分享手中的存檔!!不要再當潛水客啦!
