【短篇】音之風 第十一章 陰謀負手算天下

Sword在床上，不斷的咳嗽，sonic等人趕緊過來探望，不過silver打破了當前的寧靜：「sword，我想問你，你當初那股內力是劉道長傳給你的，為甚麼你會如此虛弱？」sword虛弱的說：「那我想問你，內力只有一種用途嗎？」silver不發一語，sword又笑著說：「劉道長給我的，有兩種用途：一種是強健體魄，武功更盛，另一種是增強智計，但身子虛弱。」sword說著，又咳了兩聲，sword又說：「sonic，你不是一直想知道風雲訣嗎？」sonic點點頭，sword說：「道長手抄了一份，在他手上，現在，是翻開的時候了。」說著昏睡了過去

風雲訣的秘密，其實是卓河清當時準備送給呂百合的東西。

當時他已和天下人結仇，已有彌補之意，要派人送給她那天，她派人來殺人，而那天，是卓河清練功的弱點：十五日是練行經脈之日，當日武功全無……。

( 音天城大堂 )

「找到了！這就是風雲訣！」

「快點。」劉斌急忙拿了過來：「哎呀，妙哉妙哉！這是天下最強的『十絕天下陣』！他終於會了。」quartz好奇的問：「這是什麼陣法？」劉斌讚嘆地說：「十絕天下陣乃是天下第一大陣，須由一字長蛇陣、蜀山派兩儀微塵陣、天地三才陣、四象八卦陣、武當派大五行劍陣、六合八荒掌、全真派大北鬥陣法、太極八卦陣和九宮八卦陣。再配上七劍組成七星聚會。」tails疑惑得問道：「六劍之主都有了，但少了魚腸劍主呀！」劉斌笑著說：「你不用擔心，因為說出來，保證嚇死你！」劉斌看了看全場，清了清喉嚨：「此人正是Sun[註1 ]，上屆音天城武功榜單榜首。」音天城的俠士全倒抽了一口氣，論起sun的武功，是不下於劉道長的。劉斌又笑著說：「如今靈天維攻來，景如英，我需要你用『功越三界』去sonichub一趟。」

靈天維看著滴水時鐘：「辰時一到，即刻發兵音天城！」

劉斌站在大廳上，鼓足丹田，大喊了一聲：「今日，靈天維舉兵犯我城門，誰願與爾等殺敵？」「我等願意！我等願意！」sonic也在心裡暗暗的喊了：「我也願意。」劉斌起御劍之術，sonic一行人也從旁邊拿了一把劍。

兩邊不約而同，大喊一聲：「殺！」千軍萬馬傾出。

果真，風雲訣中記載的陣法，確實非常人能及，一千對十五萬，卻能如同十萬高手之力，shadow心想：「quartz成功了。」劉斌一邊舉劍揮舞，一邊想著：「如今此陣為應急之需，並沒有發揮出最大實力。還要sword指揮才行。」但sword正躺在床上休息，無力起身，前線的一位俠客突然衝了過來：「報告，對方不知用甚麼陣法，我方前線崩潰！」「不好！」劉斌親上城門，silver說道：「這是sword說過的五虎群羊陣！」

五虎群羊，陣如五隻虎圍一羊，全成包圍狀態，劉斌一掌「六合八荒掌」打去，仍無法傷得幾人，這時最需要的，無非是一位陣法大家的指導。

這時，一個人拿著一根使節棍，正如當年一樣，一個拐杖走遍天下。

Sword蹣跚地起身，說道：「一轉三，三轉五，五轉七，七轉九，九宮之勢，七星圍繞。」sword此時雖然虛弱，神情卻是信心十足，sonic趁著應敵空檔之際，轉頭對sword說：「兵臨城下，你還有心情看戲呀！」sword冷靜地看了看陣勢，「背水一戰，能退敵十萬。」果真，雖有五虎群羊一陣，卻能用一千和十五萬勢均力敵，突然一柄箭從sword身邊呼嘯而過，sheath縱身一躍，把箭從上往下丟，sword笑著說：「你下去幫那些人打壞人，可以嗎？」sheath嗯了一聲。

此時，前方傳來一聲爆響：「援軍來了！」

一隊從soleanna來的部隊，敵軍瞬間被擊潰，一個時辰後，千百俠士的努力，終於達成了，sword感慨的說：「如今，已經沒有人拔的了這座音天城了。」

「甚麼？為甚麼會輸？」靈天維在長廊上大吼著，林寒慌張地說：「那隊soleanna的軍隊不聽使喚，前去破陣啦！」靈天維覺得不對勁，這軍隊不就是你管的嗎？
但來不及了，林寒兩枚冰針轉瞬飛了出去，刺穿靈天維的兩個手掌，林寒冷冷的笑道：「這是林家獨有的『冰鳳穿魂針』，武功再好，勢必會被廢去一身武功，可惜呀！」靈天維一劍飛過去，只聽見噹的一聲，劍就飛回了靈天維手上，皇甫中一劍「天道劍」刺去，和靈天維的「天泉劍法」相交，劍光四濺。

「報告宗主，人抓到了。」sword笑著說：「道長，你要提進來嗎？」劉斌點了點頭：「不用了，我親自去看他。」

( 牢房中 )

靈天維一臉頹廢，當他看到劉斌和他身後的人時，怒吼一聲：「sword，你到底是什麼人？竟要把我逼至如此地步！」sword冷笑著說：「望你能反省，師尊的話，你忘了嗎？」靈天維跌坐在地上，sword又說：「此時回頭，還來得及。璇雲山莊的人不會怪你的。」靈天維把一片東西給了劉斌，突然大笑了起來：「劉斌，我在靈御與你交戰如此多年，你還是贏了，可你永遠不會懂，師尊的異空轉生，到底是做什麼的！」頭一撞柱子，死了

劉斌看著他，緩緩地說：「只要曾持有天地陰陽符其中一塊，將來十年後方可再生，希望他變一個好人吧！」但sword在低頭沉思：「異空轉生，異空轉生……。」
突然，璇雲山莊那一幕在腦海中閃過：「真正的幕後黑手，各位道長一定認識。」「誰？」靈天維屠莊攻城，諸葛後人異空轉生，屠殺璇雲山莊那群人，這一切的一切，全部串連起來……。「我知道了，」sword說道：「紅顏隱士──石玉庭。」

[ 註一 ]：此處的sun為willzano的角色，對方已同意，並在此感謝客串。
