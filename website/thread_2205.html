在此先說明,技能升Lv是不會增加使用PP的And 味方=我方,因為本人習慣說味方請原諒

Sonic集 :
Axe Kick 敵方單體 1Lv: 2 x 150% (攻擊力) 2Lv: 2 x 175% 3Lv: 2 x 200% 使用pp : 4
特殊效果 : 沒有
人物:Sonic

Whirlwind 敵方全體 1Lv : 100% 2Lv :110% 3Lv: 125% 使用pp: 6
特殊效果: 不可能Miss,造成風擊 即速度下降(逃走速度)
人物:Sonic

Blue Bomber 敵方單體 1lv : 2 x 200% 2lv: 2 x 225% 3lv : 2 x 250% 使用PP : 4+4=8
特殊效果: 一定機會中 衰弱 使攻擊力下降
人物:Sonic and Tails

Fast ball 敵方單體 1lv 3x175% 2lv 3 x 200% 3lv 3x225% PP: 5+5=10pp
特殊效果 :一定機會 中遲鈍 使敵方閃避率下降
人物:Sonic, Amy

Triple Tornado :敵方全體 1lv 3 x 150% 2lv : 3x175% 3lv 3x225% pp: 5+5+5=15pp
特殊效果:不可能Miss , 一定機會 中遲鈍 使敵方閃避率下降 , Element : Wind
人物:Sonic,Tails and Knuckles

Hail Storm : 敵方單體 1lv : 7 x 175% 2lv 7 x 200% 3lv: 7x225% pp:6+6+6+6=24pp
特殊效果:一定機會 暈眩(可以說是ZZZ即Sleep)限定回合不能做任何動作
人物:Sonic,Tail,Kunckies and Amy
這招攻擊高的可以秒boss (差不多) 可惜使用pp量高的要命

人物升級加點: 全力加Attack(命中)
