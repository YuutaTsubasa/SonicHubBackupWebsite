【短篇】音之風最終回 第十二章 昔日劍神一笑間

Sword看著被劉斌燒完的木片：「今日，是復仇之時，是救天下之時。」sword坐在壁爐前沉思：「如果她還沒察覺，就不會找出我是卓河清的秘密，現在也就安定。」

事情總是沒有這麼如意。

隔天，一個收密報衝進的進來說：「宗主，不好了！」「不要再說不好了！我都聽膩了，什麼事？」「卓河清在我們音天城裡。」「……。( 不好。 )」sword故作放鬆地說：「區區一個卓河清，能做什麼？」此時sonic，tails，amy等人好奇地跑來問：「司空舵主，卓河清是誰？」司空舵主激動的說：「你們不知道呀？當年卓河清是一位武功第一的大俠客，與天下人結私仇無數，不過結甚麼仇，當事人恐怕不知道，你們一定有和他結仇，自己卻渾然不知。」sonic說：「那好，我們這就去抓他。」待sonic他們走了後，sword嘆息道：「石玉庭先出手了對吧？兩三天過後，如果他們發現，二話不說就會殺了我，對吧？」「是的。」「那好，我就讓她知道，真正的卓河清，有多少勇氣和天下人作對。」

Sword將sonic一行人叫進房間，把一疊紙重重的放在桌上：「我查出來卓河清了，人已經找到了。」sonic急著說：「那好，我要狠狠地……。」「且慢。」sword做了一個手勢：「你們不會想知道真相的。」shadow冷冷地說：「看與不看，有何差別？」sword嘆了口氣：「silver，記得blaze如何犧牲的嗎？她可以不用犧牲的。」silver驚訝地跌坐在地上，sword又說：「tails，你的我就不用說了吧？」tails望著地上，一語不發，sonic馬上抓住sword，將他往將牆邊丟去：「那你呢，你對我做了甚麼？」sword撐起虛弱的身子：「卓某向來只結私仇，石玉庭這個陰謀，我只能先刺破，你們如果不相信我，現在殺了我，沒關係。」sonic將旁邊的劍拔出，sword說：「目前石玉庭要用異空轉生，只有我有計策對付她。」，sonic
用顫抖的手拿著劍指著sword，心意已決，手鬆開，劍噹的一聲落在地上：「那……那……你事成後，不准出現在我們的世界作亂，站在天下人的面前，明白嗎？」sword虛弱的坐在椅子上：「好，我答應你。」

我不得不這麼做……。

( 大堂 )

Sword緩步走向前座，朗聲說道：「請魂越掌門傳下天地陰陽符。」劉斌端起那個符，交給了sword，sword說道：「謝掌門賜符，我定當用掌門的內力與符，就天下蒼生，除天下妖魔。」

Sword走後，sonic哼了一聲：「你為什麼要幫那個十惡不赦的傢伙？」劉斌望了望天：「他此番一去，怕是回不來了。」sonic疑惑道：「你這是什麼意思？」劉斌搖搖頭：「他這次去石玉庭那，是用上自己虛弱的性命，救天下人一命，就算天下人不知……。」sonic心裡一陣複雜，他要結仇，要救天下，甚麼意思？但聽到那句「天下人不知」，tails點頭說：「我願傾力相助。」，不一會兒，shadow也說：「我等願意。」「我願意。」「我也願意。」的聲音此起彼落，sonic搖著頭說：「sword，這是我最後一次幫你了。」

Sword一步一步地走進深山，就像當時，一步一步走進願意幫他的懸雲山莊，一步一步走進靈天維的地盤。走進了小涼亭中。

Sword看著這片野草叢生：「這是當年的璇雲山莊吧？」一個女生的聲音傳了進來：「好眼力，音天城城主。」sword從懷裡掏出一個東西：「諾，這是天地乾坤符，給你吧。」石玉庭拿著那塊符，看著sword：「明劍石，你太天真了。」sword問道：「我天真什麼？」「你可真是武林中人，外面一群俠客做甚麼的？送死的嗎？」sword看著城下，現在他知道，全天下的性命在他手上：「你可知道，你敢對劍某如何，我認識多少江湖人，會來算帳？」石玉庭呵呵一笑：「卓河清，我沒在裝傻，你知道這符要如何用？」sword說道：「起陣毀天滅地，然後用異空轉生，將餘黨掃蕩乾淨，沒錯吧？」石玉庭說道：「周圍只要有一點生靈，那怕是一人就會讓陣法啟動。霎時天崩地裂，你一個城主能做甚麼？」「看戲喝茶嘍。甚麼時候起陣，我等得不耐煩了。」「現在！」陣已起，如今誰也阻止不了。

一片安靜。

率先打破寧靜的石玉庭疑惑地問道：「怎麼沒啟動？我不是擺好了嗎？」sword看了看外面：「我等了一十三年，你這個只能開一次的陣還是沒用了。」石玉庭往外一看，外面哪有俠士？一片荒漠。

Sword緩緩地站起身說：「二月初三，你轉生了一干人，直攻璇雲山莊。」「四月十二，你又轉生了一些人，在魂越搗亂。」「五月二八，你勾結靈天維，準備毀了音天城，毀了三界。」「而今天，我阻止了你。」石玉庭提著劍，往sword衝去，跌倒在地上，石玉庭哈哈大笑：「你來這裡，到底是做甚麼的？要名要利，要復活璇雲山莊，要復活呂百合，還是殺了我？」sword仰望著天，喃喃的說：「燒了這陣，順道一提，劉斌和東方風靈很想你，去和你東方師傅道個歉，足矣。」

石玉庭一臉驚恐，他當年被趕出魂越，正是因為想著劉斌。

( 閣樓裡 )

「師父和我想呀，這是他最後悔的事了，妳就道個歉吧。」石玉庭一步一步走進房間，那個人，那個人，正是當年前任掌門──東方風靈。東方風靈還是那麼慈祥，石玉庭激動地跪在地上，大喊：「師傅，我想學武功，我想學武功！」，東方風靈呵呵一笑，tails天真地問：「石玉庭怎麼不動了？」劉斌搖搖頭，原來她說完的剎那，已經自盡。

「這是石玉庭生前寫的，曾異空轉生的人，全部除掉，不要留活口。」sword跟quartz再三叮囑，sword舉起毛筆，在一些人的名字前標了記：「這些人留著，派一些音天城的俠客援護一下，武功好的，不用了。」sword鬆了一口氣：「總算是塵埃落定了。」

( 三個月後，sonic回到他們的地方，斗轉星移……。)

「報。」「捷報。」「報告宗主！」quartz走進來大堂說：「你說現在該如何？」「我要去sonichub一趟，城的事，都給你了呀！」此時，一個舵主衝進來說：「報，sonic他們來訪。」sonic不等那位舵主說完，衝進來說：「你要走了？去sonichub？」sword點點頭：「sonichub有一些活動，我想去看看。」tails大喊：「胡說，你是不是要去辦事？你有道長的神功『功越三界』，加上chaos emerald，你有多久的體力？」「一個月，一個月就能辦完。」sonic說：「會有人發現嗎，你知道你倒下的時候，誰會發現？」 sword做了一個伸手的動作，sonic無奈地搖搖頭：「好，我的chaos emerald給你就是了。」amy哭著說：「你……你不必這樣的。」sword看了看音天城，頭也不回的走了。

荒漠起塵沙，風起時，那雲散時呢？

( 二年後 )

Quartz寫著音天城的榜單，想著：「公子榜榜首要寫誰呢？」突然，鎮上一隻藍色的刺蝟，一邊衝一邊喊：「卓河清！卓河清！」
